Detailed Description

For a histogram request entry, the following options are available. Upright entires are mandatory.

  • name: The name of the generated histogram plot results.
  • xaxis: The title of the x axis. Leaving blank if not specified.
  • yaxis: The title of the y axis. Defaults to "Events" is not specified.
  • units: The units of the x axis. Leaving blank if not specified. Notice that this entry will also affect the y axis display title (see the Pad1D::SetHistAxisTitle for more details).
  • filekey: The histogram "key" to get the histogram for each processes.
  • type: What to display in the secondary pad: Allowed values:
    • "simple": No secondary pad.
    • "ratio" (default): The ratio between the data and the background is displayed.
    • "pull" : The pull of the data vs the background uncertainty is displayed.
  • logy: Whether or not to display the y axis in the main pad in log scale. Default to false.
