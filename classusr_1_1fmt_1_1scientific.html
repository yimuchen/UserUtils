Detailed Description

Scientific notation of a measurement with uncertainty, with the central values and uncertainties scaled by a common exponent such that the mantissa of the central value is inline with the common definition of the scientific notations. If precision is set to negative for a measurement, the precision would be automatically determined such that the larger uncertainty displays at least two significant digits.

◆ SetExponent()

void usr::fmt::scientific::SetExponent ( )
private

If the central value is none-zero, then extracting the exponent of the central value such that \(1 < |\mathrm{central}_\mathrm{man}| < 10\); If the central value is zero, then the exponent is determined by the larger of the uncertainties, if it is non-zero. If everything is zero, then the exponent is not set.

◆ SetPrecision()

void usr::fmt::scientific::SetPrecision ( )
private

If the central value is smaller than the larger uncertainty, then at most the precision is 1 (if the uncertainty after the exponent has been factored out smaller than 10). if the central value is larger than the larger uncertainty, the precision is set such that two significant digits of the larger uncertainty is displayed.
