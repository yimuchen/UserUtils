Detailed Description

While it is unambiguous to defined a rigid pad–canvas relation, it could be inconvenient to the user who just wants to write a plot an figure. This package provides classes that provides default layouts that are commonly used in CMS analysis, as well as providing interface short hands that might be more familiar to ROOT uses (e.x. plotting on a canvas).
