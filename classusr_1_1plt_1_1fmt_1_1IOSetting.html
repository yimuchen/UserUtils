usr::plt::fmt::IOSetting Class Reference
PlotUtils » Standardized Plotting routine

Detailed Description

For the declaring common io settings, the following options are possible. None of the entires are mandatory, all entries will default to an empty string if not specified.

  • input prefix: The input prefix to add to all files. Think the working path to where the files are stored.
  • key prefix: A common prefix to add to all histogram keys. Think the TDirectory used for storing histogram files (like those generated in TFileServices in CMSSW.)
  • output prefix: Output prefix to save generated plot files. Think specifying a path to where the outputs should be passed.
  • output postfix: Output postfix for the plot files before the file extension.
