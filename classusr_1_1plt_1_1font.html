Detailed Description

Since font setting are more commonly copied around, implemenation of font conversion functions would be a class to enable storage of variables.

Fonts in ROOT (when using precision greater than 3) are defined in pixels, which, when used together with the ROOT DPI of 72, makes it the common unit of pts.
