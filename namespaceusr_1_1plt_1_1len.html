Detailed Description

Plotting length in ROOT is defined using pixels (usually), with the conversion of pixels into printable formats using the old standard of 72 DPI.

Here we provide functions for convert from printing units (inches, mm... etc), to the standard ROOT length units, as well as commonly used dimension in CMS documents.
