Detailed Description

Plotting length in ROOT is defined using pixels (usually), with the conversion of pixels into printable formats using the old standard of 72DPI.

Here we provide functionsfor convert from printing units (inches, mm... etc), to the standard ROOT length units, as well as commonly used dimension in CMS documents.
