Detailed Description

For defining a group of processes, the following options are available. Upright entries are mandatory.

  • display: The string to display for the process group in the plots, accepts ROOT flavored Latex string.
  • latex: The string to display for generation of latex tables, notice that backslashes will need to be doubled up for "literal backslashes". If not specificed, it will assume the same value of "display".
  • color: The color to display plots for this particular group. The string can either be a standard hex code ("#012ABC"), or any color name listed in the usr::plt::col namespace ("black", "midnight"...etc). Defaults to "black" is not otherwise specified.
  • processes: An array of processes declaration. See usr::plt::fmt::Process for a complete statement of their options.
