TMatrixD usr::DecompCorvariance ( const TMatrixDSym &  m)

In the case that the input matrix is not positive definite, this function will a diagonal matrix that is just the square root of the diagonal components of the original covariance matrix. The function is essentially the TDecompChol::Decompose method.
