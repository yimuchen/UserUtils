Detailed Description

Author
Yi-Mu "Enoch" Chen

Attempts to match a reco::Candidate object to a trigger object in a list of trigger objects with a specified trigger within a given cone size (defaults to 0.1).

The triggerNames object required should be extracted directly from the edm::Event and edm::TriggerResults class ese . The requested trigger (reqtrigger) and requested label (reqlabel) could have globbing syntax (e.x. HLT_IsoMu27_v* for multiple versions)
