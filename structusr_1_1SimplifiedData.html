usr::SimplifiedData Struct Reference

Detailed Description

Additional helper routines are used for caching anc calculating the numbers required for performing the Kolmogorov-Smirnov test. One in particular is the "EffectiveNum" method, as the number of data points is required for the calculation of the KS test, the "number of data points" for weighted data sets would follow the "sum squared over sum of squared" approach:

\[ N_\mathrm{eff} = \frac{\left(\sum w_i\right)^2 }{\sum w_i^2} \]

The data set is also simplified to a list of doubles (with weights), with the list ordered to make it simpler to calculated the KS test.
