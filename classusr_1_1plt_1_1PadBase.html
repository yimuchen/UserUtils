Detailed Description

This class and it's children are to be directly owned by the Canvas class, enforcing a single layer pad–canvas hierarchy, while also providing the function PlotObj to ensure that a certain object is plotted onto a specified pad.

There are an embedded TLatex object to assist with writing on the Pad in a uniform font setting to the parent canvas. As well has function to help with calculating the effective size of the Pad.

Note that this functions would have its constructor protected, so all creation of this class would only be possible through a parent canvas object.
